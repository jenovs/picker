Live version: https://jenovs.github.io/picker/

## Tech used

React, styled-components.

Jest and testing-library for testing.

## Run locally

Clone the repo and run

```
npm i
```

to install packages.

Use

```
npm start
```

to open the app in the browser.

## Testing

Run tests with

```
npm t
```

or

```
npm run test:coverage
```

to get coverage report.
